## 💻 **Step 3: Use Command Line Tools**

### **🚀 Platform Management:**
```bash
# Set up the complete M365 platform
python -m smarthaus_cli setup-platform

# Check platform health
python -m smarthaus_cli health

# View organization info
python -m smarthaus_cli org
```

### **📋 Project Management:**
```bash
# Create a new project
python -m smarthaus_cli create-project "Project Name" --description "Description"

# List all projects
python -m smarthaus_cli list-projects

# View project details
python -m smarthaus_cli list-projects
```

### **🔍 Repository Analysis:**
```bash
# Analyze any git repository
python -m smarthaus_cli analyze-repo /path/to/repo --output-file analysis.json

# Create project template from repository
python -m smarthaus_cli create-template /path/to/repo --template-name "Custom Template"
```
