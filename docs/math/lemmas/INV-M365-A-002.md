# Lemma: Persona is exactly Admin or User

-----------------------------------------
**Lemma ID:** LEM-M365-A-002-01

**Invariant:** INV-M365-A-002

**Derived from:**
- Section 1: Sets and Domains (\(\mathcal{P} = \{\mathsf{Admin},\mathsf{User}\}\))
- Section 2: Instruction Structure

**Statement:**

For all \(x = (p, i, a, \mathit{params}) \in \mathcal{X}\), \(p \in \mathcal{P}\) and \(\mathcal{P} = \{\mathsf{Admin},\mathsf{User}\}\). Therefore \(p \in \{\mathsf{Admin},\mathsf{User}\}\) with no other value possible; persona is neither absent nor ambiguous.

**Assumptions:**

- Phase 1 defines \(\mathcal{P} = \{\mathsf{Admin},\mathsf{User}\}\) with the explicit statement "Exactly two personas; no other values" (Section 1).
- Every instruction satisfies \(x = (p, i, a, \mathit{params})\) with \(p \in \mathcal{P}\) (Section 2).

**Proof outline:**

1. \(\mathcal{P}\) is defined as the set \(\{\mathsf{Admin},\mathsf{User}\}\).
2. By set equality, every element of \(\mathcal{P}\) is either \(\mathsf{Admin}\) or \(\mathsf{User}\), and there are no other elements in \(\mathcal{P}\).
3. For every \(x \in \mathcal{X}\), the first component \(p\) satisfies \(p \in \mathcal{P}\) by definition of \(\mathcal{X} = \mathcal{P} \times \mathcal{I} \times \mathcal{A} \times \mathcal{P}\!\mathit{ar}\).
4. Therefore for every instruction, \(p \in \{\mathsf{Admin},\mathsf{User}\}\).
5. Absence or ambiguity of persona would correspond to \(p\) not being a single element of \(\mathcal{P}\); that is excluded because the instruction is an element of the product and the first coordinate is uniquely determined.

**Edge cases:**

- No third persona exists: \(\mathcal{P}\) is defined by enumeration; no other symbol is in \(\mathcal{P}\).
- "Ambiguous" persona would require the spec to allow multiple or partial values; the spec defines a single element of \(\mathcal{P}\) per instruction.

**Conclusion:**

The definition of \(\mathcal{P}\) and of \(\mathcal{X}\) implies that the persona component of every instruction is exactly one of Admin or User. Hence INV-M365-A-002 holds.
