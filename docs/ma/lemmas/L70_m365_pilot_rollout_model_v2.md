# L70 — M365 Pilot and Rollout Model v2

## Lemma

The rollout may be treated as deterministic only when: (1) the rollout contract exists with four phases and five governance rules, (2) three sequential waves are defined with acceptance gates, and (3) pilot acceptance is required before expansion.

## Proof Sketch

1. If four rollout phases exist, the path is structured.
2. If three waves advance sequentially, rollout is ordered.
3. If pilot acceptance gates expansion, overclaiming is prevented.
4. Therefore rollout is deterministic and bounded.
