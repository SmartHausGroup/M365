# SMARTHAUS Color Palette

**Source:** Official swatch — `public/assets/SMARTHAUS_COLOR_PALETTE.png`

Use these hex codes for SMARTHAUS branding: M365 PyQt6 dashboard, web UI, docs, and any app that carries the SMARTHAUS brand.

---

## Palette (18 colors)

### Row 1 — Primary blues & neutrals

| Hex | Use |
|-----|-----|
| `#43708F` | Medium royal blue — accents, links, primary actions |
| `#A4A2A2` | Light–medium gray — secondary text, borders |
| `#000000` | Pure black |
| `#070C10` | Very dark teal — darkest background |
| `#0F1920` | Very dark navy — dark background |
| `#162530` | Dark navy — header background, panels |

### Row 2 — Blues (mid–dark)

| Hex | Use |
|-----|-----|
| `#1E3240` | Deep navy — surfaces, cards |
| `#253E4F` | Muted dark blue — borders, dividers |
| `#2D4B5F` | Medium–dark blue — hover, secondary surfaces |
| `#34576F` | Teal-blue — accents |
| `#3C647F` | Slate blue — accents |
| `#162530` | Dark navy (also row 1) — header/panel accent in mid palette |

### Row 3 — Lights & grays

| Hex | Use |
|-----|-----|
| `#E1DCDC` | Off-white / cream — light backgrounds, primary text on dark |
| `#A8A8A8` | Medium–light gray — muted text |
| `#8F888F` | Medium–dark gray — secondary text |
| `#595599` | Dark gray (purple-gray) — borders, subtle UI |
| `#595599` | Dark gray (purple-gray) — second swatch in palette |
| `#2E2E2E` | Very dark gray — dark UI surfaces |
| `#1B1B1B` | Black / near-black — dark backgrounds |

---

## Suggested roles (e.g. PyQt6 dashboard)

| Role | Hex | Notes |
|------|-----|--------|
| Window / main bg | `#0F1920` or `#162530` | Deep navy |
| Header / title bar | `#162530` | Matches swatch header |
| Surface / cards | `#1E3240`, `#253E4F` | Blues from row 2 |
| Primary text (on dark) | `#E1DCDC` | Cream |
| Secondary text | `#A4A2A2`, `#A8A8A8` | Grays |
| Accent / primary button | `#43708F`, `#4A6680` | Royal / medium blue |
| Border | `#253E4F`, `#595599` | Dark blue or gray |
| Input / field bg | `#1E3240` or `#253E4F` | Slightly lighter than window |
