# Contributing

- Use Python 3.11+
- Install dev deps: `make install-deps`
- Run `make lint` and `make test` before pushing
- Open PRs against `main` with clear scope and checklist

## Branching
- `main`: protected, CI required
- feature branches: `feat/...`, `fix/...`, `chore/...`

## Commit Style
- Conventional commits (`feat:`, `fix:`, `chore:`)

## Security
- Never commit secrets; use Key Vault and env vars
